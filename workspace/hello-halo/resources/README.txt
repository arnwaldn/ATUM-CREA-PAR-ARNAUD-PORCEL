====================================
    ATUM CREA - Installation Guide
====================================

Thank you for downloading ATUM CREA!

-----------------------------------
Installation
-----------------------------------
1. Drag ATUM CREA.app to the "Applications" folder
2. Double-click ATUM CREA.app to launch

-----------------------------------
"Cannot be opened" warning?
-----------------------------------
This is a normal macOS security prompt. Follow these steps:

Option 1 (Recommended):
1. Right-click on ATUM CREA.app
2. Select "Open"
3. Click "Open" in the dialog

Option 2:
1. Open "System Settings" > "Privacy & Security"
2. Scroll down to find the ATUM CREA message
3. Click "Open Anyway"

You only need to do this once.
