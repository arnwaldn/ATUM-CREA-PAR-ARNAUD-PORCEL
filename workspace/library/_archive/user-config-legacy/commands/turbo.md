# Mode TURBO Active

## WORKFLOW AUTOMATIQUE

### PHASE 1 - RECHERCHE (Context7 + Exa)
Avant de coder, utilise obligatoirement:
1. **Context7** pour obtenir la documentation a jour du framework (Next.js 15, React 19, etc.)
2. **Exa** ou **Tavily** pour trouver des exemples similaires et patterns

### PHASE 2 - ARCHITECTURE
1. Propose une structure de fichiers optimale
2. Si database necessaire, prepare le schema
3. Identifie les composants principaux

### PHASE 3 - IMPLEMENTATION
Regles strictes:
- **TypeScript strict** - Pas de `any`, types complets
- **shadcn/ui** pour tous les composants UI
- **Code complet** - Pas de TODO, pas de placeholders, pas de "..."
- **Imports corrects** - Verifie chaque import
- **Next.js 15 App Router** - Patterns actuels

### PHASE 4 - VALIDATION
Avant de terminer:
- Verifie que tous les types sont corrects
- Verifie que tous les imports existent
- Verifie la coherence du code
