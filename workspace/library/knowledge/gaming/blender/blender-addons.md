## Quick Install (Blender 5.0)

```python
# Via Blender Preferences
Edit > Preferences > Add-ons > Install > Select .zip

# Via Script
import bpy
bpy.ops.preferences.addon_install(filepath="path/to/addon.zip")
bpy.ops.preferences.addon_enable(module="addon_name")
```
