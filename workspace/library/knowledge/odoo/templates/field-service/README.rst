Features
========

Intervention Management
-----------------------
* Full intervention lifecycle (Draft → Scheduled → In Progress → Done)
* Multiple intervention types (Repair, Maintenance, Installation, Emergency, Inspection)
* Priority management with SLA tracking
* Calendar integration for scheduling
* Kanban board with drag-and-drop
* Parts and labor tracking
* Customer and technician signatures
* PDF intervention reports

Technician Management
---------------------
* Technician profiles with skills and certifications
* Real-time availability status
* Skill levels (Beginner, Intermediate, Expert)
* Certification tracking with expiry alerts
* Work schedule configuration
* Hourly and overtime rates

Equipment Tracking
------------------
* Customer equipment registry
* Equipment types with required skills
* Serial number and warranty tracking
* Preventive maintenance scheduling
* Maintenance alerts and notifications
* Full intervention history per equipment

Service Contracts
-----------------
* Multiple contract types (Maintenance, Support, Warranty, Full Service)
* Coverage levels (Basic, Standard, Premium)
* Included interventions quota tracking
* SLA response and resolution times
* Parts, labor, and travel coverage options
* Auto-renewal configuration
* Contract expiry alerts

Configuration
=============

Initial Setup
-------------

1. **Intervention Types**: Configure your intervention types with default durations and SLA times
   - Navigate to Field Service > Configuration > Intervention Types
   - Set up types like Repair, Maintenance, Installation, Emergency

2. **Skills**: Define technician skills and certification requirements
   - Navigate to Field Service > Configuration > Skills
   - Mark skills that require certification

3. **Equipment Types**: Configure equipment categories
   - Navigate to Field Service > Configuration > Equipment Types
   - Set default maintenance intervals and required skills

4. **Technicians**: Register your field technicians
   - Navigate to Field Service > Resources > Technicians
   - Add skills and certifications for each technician

User Roles
----------

The module provides three security groups:

* **Field Technician**: Can view assigned interventions, update status, log time
* **Dispatcher**: Can create and assign interventions, manage schedules
* **Field Service Manager**: Full access to all features, reports, and configuration

Usage
=====

Creating an Intervention
------------------------

1. Navigate to Field Service > Operations > Interventions
2. Click "Create" to open a new intervention form
3. Select the customer and intervention type
4. Add problem description and location details
5. Save as draft or schedule immediately

Scheduling an Intervention
--------------------------

1. Open a draft intervention
2. Click "Schedule" button or use the Schedule Wizard
3. Select date, time, and technician
4. The intervention will appear on the calendar

Completing an Intervention
--------------------------

1. When on-site, click "Start" to begin the intervention
2. Log parts used and time entries
3. Add work performed description
4. Click "Complete" when finished
5. Capture customer signature if required

Managing Contracts
------------------

1. Navigate to Field Service > Contracts > Service Contracts
2. Create contracts with specific coverage levels
3. Link customer equipment to contracts
4. Monitor intervention usage and SLA compliance

Reports
-------

* Intervention Report: PDF summary of intervention with parts, labor, and signatures
* SLA Compliance: Track response and resolution time compliance
* Technician Performance: Interventions completed, average duration, ratings

Integration
===========

This module integrates with:

* **Calendar**: Interventions appear as calendar events
* **Account**: Generate invoices from billable interventions
* **HR**: Link technicians to employees
* **Mail**: Activity and message tracking
