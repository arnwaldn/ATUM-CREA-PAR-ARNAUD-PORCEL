Features
========

Table Management
----------------
* Floor plan configuration with multiple areas
* Visual table status (Available, Occupied, Reserved, Cleaning)
* Table shapes and capacity management
* Drag-and-drop table positioning
* Quick table status updates

Order Management
----------------
* Full order lifecycle (Draft → Confirmed → Preparing → Ready → Served → Paid)
* Order types: Dine-in, Takeaway, Delivery
* Split orders and merge tables
* Item modifications and special requests
* Kitchen ticket printing
* Customer bill generation

Menu Management
---------------
* Menu categories with hierarchy
* Menu items with pricing and cost tracking
* Dietary information (Vegetarian, Vegan, Gluten-free, Allergens)
* Item options (Size, Cooking level, Sides, Extras)
* Ingredient tracking for recipes
* Time-based availability (Breakfast, Lunch, Dinner)

Kitchen Display
---------------
* Real-time order display for kitchen staff
* Order prioritization
* Item status tracking
* Printer routing by station (Kitchen, Bar, Dessert)

Reservations
------------
* Online and phone reservation management
* Table assignment
* Party size tracking
* Occasion tagging (Birthday, Anniversary, Business)
* Automatic reminder notifications
* No-show tracking

Staff Management
----------------
* Staff profiles with positions
* PIN-based authentication
* Shift tracking
* Performance metrics (Orders, Sales, Tips)
* Hourly rate management

Payment Processing
------------------
* Multiple payment methods (Cash, Card, Mobile)
* Split payments
* Tip handling
* Change calculation
* Invoice generation

Configuration
=============

Initial Setup
-------------

1. **Floors**: Configure your restaurant floors/areas
   - Navigate to Restaurant > Floor Plan > Floors
   - Create floors (Main Dining, Terrace, Bar, etc.)

2. **Tables**: Add tables to each floor
   - Navigate to Restaurant > Floor Plan > Tables
   - Set capacity and position for each table

3. **Menu Categories**: Organize your menu structure
   - Navigate to Restaurant > Menu > Categories
   - Create categories (Starters, Mains, Desserts, Drinks)

4. **Menu Items**: Add your dishes and beverages
   - Navigate to Restaurant > Menu > Menu Items
   - Set prices, dietary info, and preparation time

5. **Staff**: Register your team
   - Navigate to Restaurant > Staff
   - Add positions and PIN codes for POS access

Kitchen Printers
----------------

Configure printers for different stations:

* **Kitchen**: Main food preparation
* **Bar**: Beverages and cocktails
* **Dessert**: Sweet items and coffee

User Roles
----------

The module provides four security groups:

* **Waiter**: Take orders, process payments, manage tables
* **Kitchen Staff**: View and manage kitchen orders
* **Host/Hostess**: Manage reservations and table assignments
* **Manager**: Full access to all features and reports

Usage
=====

Taking an Order
---------------

1. Select a table from the floor plan
2. Click "New Order" or open existing order
3. Add items from the menu
4. Apply modifications if needed
5. Confirm order to send to kitchen

Processing Payment
------------------

1. Open the order from served state
2. Click "Payment" button
3. Select payment method
4. Enter amount and process
5. Print receipt if needed

Managing Reservations
---------------------

1. Navigate to Reservations menu
2. Click "Create" for new reservation
3. Enter customer details and party size
4. Select date/time and preferred tables
5. Confirm reservation

Kitchen Workflow
----------------

1. Kitchen staff views orders on Kitchen Display
2. Click "Start" when beginning preparation
3. Click "Ready" when dishes are complete
4. Waiters are notified to serve

* **HR**: Staff linked to employees
